Set Up Your Daily Email Reminder

Claude can't send automated emails on its own, but you can set this up in 2 minutes using Gmail's scheduled send or Google Calendar. Here's exactly how:

1
Open Gmail and compose a new email to yourself

Send it to thespiangabriel@gmail.com. Use the subject line and body below — copy it exactly.

Subject: 🔍 Daily Job Search — Run WakaJobs Now Good morning Paul, Time to run your daily product design job search. 👉 Open your WakaJobs dashboard and hit "Run Today's Search": [Paste your WakaJobs bookmark URL here] Or use this Claude prompt directly: https://claude.ai/new --- YOUR DAILY SEARCH PROMPT --- You are my AI recruiter. Search for real, currently open product designer / UI UX designer job listings posted in the last 24–48 hours that match my profile: Name: Paul Anuge Role: Product Designer (entry–mid level, 1–3 yrs exp) Skills: Figma, UX research, wireframing, prototyping, usability testing, Canva, HTML/CSS basics Industries: Fintech, Healthtech, Edtech, SaaS Location: Nigeria (Ibadan/Lagos) — open to fully remote roles and EMEA-eligible remote positions Previous experience: Umba Pay (loan app UX, social media design), Bluecoral Healthcare (digital marketing), Toast Dating (community marketing) Find at least 15 matching roles. Present as a dashboard table with: Job Title | Company | Location | Salary Range | Required Skills | Application Link | Match Score (out of 100). Sort by Match Score. Group into Tier A (75–100), Tier B (55–74), Tier C (30–54). --- Good luck today! 🚀
2
Schedule it to repeat daily using Gmail + Google Calendar

Gmail doesn't have native recurring send, so use Google Calendar instead: create a daily event at your preferred morning time (e.g. 8:00 AM), paste the prompt in the event description, and set a email notification so it pings you. That's your daily trigger.

Steps: 1. Go to calendar.google.com 2. Create new event → set time to e.g. 8:00 AM daily 3. Set recurrence: "Every day" 4. Add notification: "Email" → "0 minutes before" 5. Paste the Claude prompt above in the description 6. Save Every morning at 8 AM, Google Calendar sends you an email with the prompt ready to paste into Claude.
